Webbender

Bend the web to your will.

A feature-rich bookmarklet built for quick UX reviews. Open any page, click the bookmark, and instantly test copy edits, cleanup ideas, dialog behavior, and temporary themes.

Install

bookmark Webbender

Drag the button above to your bookmarks bar or click to copy the bookmarklet code to your clipboard.

What you can do

edit

Edit Text

Enable design mode and rewrite text directly on production-like pages without opening DevTools.

delete

Remove Elements

Fast cleanup checks. Remove noisy elements and verify layout/readability in seconds.

text_fields

Typography Override

Apply temporary system or custom web fonts to evaluate legibility and brand alignment.

drag_pan

Grab & Move

Click and drag to reposition objects freely around the page for rapid layout experiments.

palette

Color Themes

Switch to Dark, Light, or Sepia themes globally across the page to ensure accessibility.

forum

Dialog Helpers

Trigger alert, confirm, and prompt dialogs on-demand for comprehensive user flow testing.
