Bookmarklet-first page editing, without the extension overhead.

Self-contained bookmarklet

Bend a page without installing anything.

Webbender gives you a floating control panel for editing text, removing elements, changing typography, and testing dialog behavior. All code is embedded directly in the bookmark — no external requests, no Content Security Policy conflicts.

Quick edits Toggle design mode and make page text editable in-place.
Cleanup mode Remove distracting elements with a click and clear hover highlighting.
Self-contained All code is embedded directly in the bookmark — no external script fetch, no CSP issues.

What it does

  • Edit text directly on the page using design mode.
  • Remove distracting elements without opening devtools.
  • Apply temporary fonts and themes for quick design checks.
  • Trigger alert, confirm, and prompt dialogs for UI testing.

Why this approach

The old loader injected an external <script> tag to fetch the payload, which was blocked by Content Security Policy on most sites. All code is now embedded directly in the bookmark, so no network request is ever made and CSP cannot interfere.

How to use

  1. Open any website.
  2. Click your Webbender bookmark.
  3. Use the floating panel to edit, remove, or style elements.

Troubleshooting

  • If clicking does nothing, recreate the bookmark from this page.
  • Some sites block injected scripts with CSP restrictions.
  • If the loader fails twice, check the browser console for the error.
Webbender is published through Firebase Hosting, so the install link stays on a stable domain and the payload can be updated without changing the bookmarklet itself.

Hosted docs: webbender.web.app and webbender-pro.web.app.
