Guided install (local CLI)

If you'd rather host the installer page on Firebase Hosting (automatic updates), follow these steps locally:

  1. Install Firebase CLI and login: npm install -g firebase-tools and firebase login
  2. Initialize hosting (choose site as public folder): firebase init hosting --project webbender-pro
  3. Deploy: firebase deploy --only hosting --project webbender-pro

CI/CD

This repository includes a GitHub Actions workflow to deploy this site to Firebase Hosting. You need to provide a Firebase service account JSON as a repository secret named FIREBASE_SERVICE_ACCOUNT_WEBBENDER.
