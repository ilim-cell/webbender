Webbender

A bookmarklet for quick page editing, element cleanup, and visual experimentation on any website.

What Webbender Does

  • Edit text directly on the page using design mode.
  • Remove distracting elements without opening devtools.
  • Apply temporary fonts and themes for quick design checks.
  • Test native alert/confirm/prompt dialog behavior.

How to Use

  1. Open any website.
  2. Click your Webbender bookmark.
  3. Use the floating panel to toggle edit/remove/theme controls.

Troubleshooting

  • If clicking does nothing, delete and re-create the bookmark from this page.
  • Some sites block external scripts with strict CSP headers.
  • Open browser console for loader errors if script fetch fails.

Guided install (local CLI)

If you'd rather host the installer page on Firebase Hosting (automatic updates), follow these steps locally:

  1. Install Firebase CLI and login: npm install -g firebase-tools and firebase login
  2. Initialize hosting (choose site as public folder): firebase init hosting --project webbender-pro
  3. Deploy both sites: firebase deploy --only hosting:sitePrimary,hosting:sitePublic --project webbender-pro

CI/CD

This repository includes GitHub Actions workflows to deploy this site to Firebase Hosting. Provide a Firebase service account JSON as repository secret FIREBASE_SERVICE_ACCOUNT_WEBBENDER_PRO.
