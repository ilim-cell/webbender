Webbender

Bookmarklet-first page editing, without the extension overhead.

Server-hosted bookmarklet

Bend a page without installing anything.

Webbender gives you a floating control panel for editing text, removing elements, changing typography, and testing dialog behavior. The installer now loads its payload from Webbender's own hosting, which avoids the MIME issues that broke the old raw GitHub/CDN path.

Quick edits Toggle design mode and make page text editable in-place.
Cleanup mode Remove distracting elements with a click and clear hover highlighting.
Hosted payload Bookmarklet code is served from `webbender.web.app`, so the browser gets executable JavaScript.

What it does

  • Edit text directly on the page using design mode.
  • Remove distracting elements without opening devtools.
  • Apply temporary fonts and themes for quick design checks.
  • Trigger alert, confirm, and prompt dialogs for UI testing.

Why the install changed

The old loader depended on raw GitHub and jsDelivr script URLs. Those responses can be served as `text/plain`, which browsers refuse to execute as script. This page now points the bookmarklet at a hosted JS file with the right MIME type.

Manual bookmarklet code

javascript:(function(){var urls=['https://webbender.web.app/bookmarklet.js','https://webbender-pro.web.app/bookmarklet.js'];var id='webbender-boot';var existing=document.getElementById(id);if(existing){existing.remove();}var index=0;function load(){if(index>=urls.length){console.error('Webbender: failed to load from all URLs');alert('Webbender failed to load. Check the browser console for details.');return;}var src=urls[index++];var script=document.createElement('script');script.id=id;script.src=src;script.onerror=function(){console.error('Webbender: failed to load',src);script.remove();load();};document.head.appendChild(script);}load();})();

How to use

  1. Open any website.
  2. Click your Webbender bookmark.
  3. Use the floating panel to edit, remove, or style elements.

Troubleshooting

  • If clicking does nothing, recreate the bookmark from this page.
  • Some sites block injected scripts with CSP restrictions.
  • If the loader fails twice, check the browser console for the error.
Webbender is published through Firebase Hosting, so the install link stays on a stable domain and the payload can be updated without changing the bookmarklet itself.
